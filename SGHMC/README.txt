# sta663-FinalProject-SGHMC
Final project for STA 663 (Duke University Spring 2018) implementing stochastic gradient Hamiltonian Monte Carlo (SGHMC).


To install the package:

Enter an environment where Python 3 is the default, then from the SGHMC directory run
$ python setup.py install

Alternatively (tested on Mac), from the SGHMC directory run
$ python3 setup.py install


To run examples:

Run script in the 'run_examples.py' file
